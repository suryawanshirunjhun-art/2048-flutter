#  2048 Game App (Flutter)

A fully functional 2048 puzzle game built using Flutter.  
This project focuses on implementing core game logic, gesture-based interactions, and responsive UI design.

---

##  Features

- Swipe-based tile movement (up, down, left, right)
- Accurate tile merging logic following 2048 rules
- Real-time score tracking system
- Restart game functionality
- Smooth and responsive user interface

---

##  Tech Stack

- Flutter
- Dart

---

##  Key Learnings

- Implemented grid-based game logic from scratch
- Handled touch gestures and user interactions in Flutter
- Managed dynamic state updates for real-time UI changes
- Debugged and optimized tile merging and movement logic
- Improved understanding of mobile app architecture

---

##  Project Purpose

This project was built to strengthen problem-solving skills and gain hands-on experience with Flutter UI development and game logic implementation.
